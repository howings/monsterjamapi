This is the API

trucksGet

Obtain information about all discontinued, existing, physical, and virtual toy monster trucks from Spin Master in company DB

URL

https://dev.spinmaster.com/v1/trucks

HTTP Method

GET

Response Type

Trucks

Parameters

  • Query: bodyLimit Integer

    The amount of trucks returned per query

  • Query: pageLimit Integer

    The pages to return truck info

trucksIdDelete

Delete a single truck by id

URL

https://dev.spinmaster.com/v1/trucks/{id}

HTTP Method

DELETE

Response Type

Truck

Parameters

  • Path: id Integer

    Truck id to delete

trucksIdGet

Obtain information of a single truck from Spin Master in company DB

URL

https://dev.spinmaster.com/v1/trucks/{id}

HTTP Method

GET

Response Type

Truck

Parameters

  • Path: id Integer

    The ID of the toy truck

trucksIdPut

Update a single truck info with id

URL

https://dev.spinmaster.com/v1/trucks/{id}

HTTP Method

PUT

Response Type

Truck

Parameters

  • Path: id Integer

    Truck id to update

trucksPost

Insert information for a single toy monster trucks from Spin Master in company DB

URL

https://dev.spinmaster.com/v1/trucks

HTTP Method

POST

Response Type

Parameters

  • Body: body Truck(Truck)
